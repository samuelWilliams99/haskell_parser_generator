DFA

Description

The DFA holds a list of DFAStates, each of which hold links to other states on reading token, similarly to how a Turing Machine DFA works. These states also hold information about the productions they correspond to, but only by index in the DFAProduction list. Various other HashMaps are also stored within the DFA to keep the structure fully self referencial and standalone.
