ShiftReduce

Description

Uses the Shift-Reduce algorithm to build a DFA from the DFA module. Shift-Reduce conflicts are automatically resolved if sufficient precedence data has been supplied, otherwise an error is thrown. Similarly, Reduce-Reduce conflicts will cause the algorithm to fail.
